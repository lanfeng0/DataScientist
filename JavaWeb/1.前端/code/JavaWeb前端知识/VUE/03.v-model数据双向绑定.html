修饰符：
在默认情况下，v-model 在每次 input 事件触发后将输入框的值与数据进行同步 (除了上述输入法组合文字时)。 你可以添加 lazy 修饰符，从而转为在 change 事件_之后_进行同步：

message: {{ message }}

如果要自动过滤用户输入的首尾空白字符，可以给 v-model 添加 trim 修饰符：

message: {{ message }}

如果想自动将用户的输入值转为数值类型，可以给 v-model 添加 number 修饰符：

age: {{ age+1 }}
