• colspan="跨的列数" 表示当前单元格跨几列
  • rowspan="跨的行数" 表示当前单元格跨几行
  • bgcolor设置表格table、行tr、单元格th/td的背景颜色
  • border设置表格边框宽度、bordercolor表格边框线的颜色
  • width、height设置表格table、tr、th、td的宽度和高度
  • cellpadding单元格的填充： 当前单元格内 文本离边框的距离
  • cellspacing单元格的间距：单元格和其他单元格的间距
  • align:表格的文字水平位置，可选值包括left、center、right。align如果是设置为table标签的属性，那么是表示整张表格对于页面水平方向，无法对单元格内的文字内容有效。
  • align如果是设置为tr/th/td标签的属性，就是单元格内文字水平位置
  • valign:表格内文字的垂直位置，tr/th/td元素的属性，valign可选值包括top、middle、bottom。
